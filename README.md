# Run and deploy your AI Studio app

- This contains everything you need to run your app locally.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
```cmd
npm install
```

2. Run the app:
```cmd
npm run dev
```
- redeploy when ready:
```cmd
npm run build
npm run deploy
```
